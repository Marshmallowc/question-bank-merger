# 示例数据说明

本目录包含了示例题库文件，用于测试题库合并工具。

## 文件说明

1. **第一章示例_习题导出.xlsx**
   - 包含3道题目
   - 题型：单选题、多选题、判断题
   - 展示了标准的中国大学题库格式

2. **第二章示例_习题导出.xlsx**
   - 包含3道题目
   - 题型：单选题、判断题
   - 用于测试多文件合并功能

## 测试方法

```bash
# 进入项目根目录
cd question-bank-merger

# 使用示例数据测试
python src/merger.py --input examples/sample_questions

# 查看输出结果
ls output/
```

## 预期输出

- Excel文件：包含6道题目
- Word文档：按章节分组的题目列表
- 控制台输出：统计报告
